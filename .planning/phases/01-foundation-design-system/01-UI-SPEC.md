## Spacing Scale

Standard scale: 4, 8, 16, 24, 32, 48, 64. All values are multiples of 4.

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding, small element gaps |
| sm | 8px | Compact gaps, card inner padding minimums |
| md | 16px | Default card padding (`px-4`, `p-4`), section gaps (`gap-4`) |
| lg | 24px | Medium gaps (`gap-6`), header padding (`px-6`) |
| xl | 32px | Layout grid gaps (`gap-8`) |
| 2xl | 48px | Hero spacing, modal padding |
| 3xl | 64px | Page-level spacing |

Brownfield exceptions (inherited from existing code, accepted for Phase 1):
- 12px — used in form fields and list items (between 8 and 16, acceptable for touch targets)
- 20px, 28px — border-radius values only, not spacing (cosmetic, not structural)

Non-structural exceptions:
- `max(env(safe-area-inset-bottom), 0px)` for mobile safe-area padding (iOS notch)
- `border-radius: 999px` for pills/chips (full border, cosmetic)

## Typography

Declared from existing `fontFamily` and `fontSize` config in `tailwind.config.js`, plus CSS definitions in `styles.css`. Consolidated to 4 sizes / 2 weights per Nyquist constraint.

| Role | Size | Weight | Line Height | Font Family |
|------|------|--------|-------------|-------------|
| Body (includes labels, code) | 14px | 400 | 1.45 | Plus Jakarta Sans (code: JetBrains Mono) |
| Label small | 11px | 400 | 1.3 | Plus Jakarta Sans (uppercase, tracking-[0.16em]) |
| Heading | 20px (text-xl) | 600 | 1.2 | Outfit |
| Display (includes h1 at 28px/3rem) | 28px+ | 600 | 1.2 | Outfit |

**Rules:**
- 4 sizes max: 11px, 14px, 20px, 28px+
- 2 weights max: 400 (body/labels), 600 (headings/display)
- Body line height: 1.45
- Heading line height: 1.2 (via `tracking-tight`)
- Letter spacing: `-0.03em` on headings, `-0.015em` on body
- Labels use uppercase with `tracking-[0.16em]`
- Code elements use JetBrains Mono at 14px/400 — same slot as Body, just different family
- Display sizes 28px through 3rem all map to the "Display" slot — same weight, same family, just size varies by context

## Color

Derived from existing `tailwind.config.js` color palette. Dark-mode only (no light mode toggle — this is a dark-first product).

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#0a0a0a` (bg) | Page background, base surfaces |
| Secondary (30%) | `#151110` (surface) through `#302825` (surface-4) | Cards, panels, sidebar, nav |
| Accent (10%) | `#10b981` (emerald) | Primary CTAs, active states, progress indicators, completed status |
| Destructive | `#fb7185` (danger) | Delete actions, error states, high priority indicators |

**Semantic colors:**
| Token | Value | Usage |
|-------|-------|-------|
| ok | `#10b981` | Success/done states |
| warn | `#f59e0b` | Warnings, medium priority, in-progress sync |
| info | `#38bdf8` | Info states, "devam" status |
| accent-2 | `#6ee7b7` | Accent hover highlights |
| accent-d | `#0d8d64` | Accent dark (pressed states) |

**Surface ink scale:**
| Token | Value | Usage |
|-------|-------|-------|
| ink | `#f7f5f2` | Primary text, headings |
| ink-2 | `#c8c0b6` | Secondary text, body copy |
| ink-3 | `#91877d` | Tertiary text, labels, hints |
| ink-4 | `#5e5650` | Disabled, muted text |

**Border scale:**
| Token | Value | Usage |
|-------|-------|-------|
| line | `#2d2523` | Default card borders |
| line-2 | `#403430` | Stronger borders, dividers |
| line-3 | `#584945` | Accent borders |

**Accent reserved for:**
- Primary action buttons (active nav tabs, main CTA)
- Completed/done status indicators
- Progress bar fills
- Sync status "senkron" badge (connected state)
- Hover text on interactive elements (`hover:text-accent-2`)
- NEVER used for destructive actions, warnings, or generic links

**Glass system (from styles.css):**
- `.glass` — translucent gradient card with backdrop blur (26px) + noise overlay
- `.glass-strong` — heavier blur (32px) + stronger border, for modals and nav bars
- `.glass-soft` — lighter blur (18px) + subtle border, for metric cards and inline elements

## Visual Hierarchy

**Primary focal point:** h1 display title ("Haftalık Planlayıcı") at 28px+/600/Outfit — draws the eye first on every screen.

**Secondary focal points (in order):**
1. Metric cards with large numbers (28px display figures in accent color)
2. Active nav tab button (accent background + shadow glow)
3. Progress bar fill (accent color, animated)
4. Priority stripe on task rows (danger/warn left-edge accent)

**Tertiary elements:** body text, labels, secondary text — recede into surface-2/ink-2 hierarchy.

**Eye flow pattern:** Header title → metric cards → content cards → nav bar

## Animation & Interaction

| Interaction | Duration | Easing | Effect |
|-------------|----------|--------|--------|
| Hover lift | 220ms | `cubic-bezier(.2, .8, .2, 1)` (spring) | `translateY(-1px)` + border/shadow change |
| Row hover | 180ms | spring | Background color shift |
| Button press | N/A | N/A | Color swap (active → accent bg) |
| Screen entrance | 300ms | spring | `fadeUp` — opacity 0→1, translateY(10px→0) |
| Side panel | 280ms | spring | `slideInRight` — opacity 0→1, translateX(18px→0) |
| Loading shimmer | 2.2s | linear | Infinite background sweep |
| Progress bar | 500ms | default | Width transition |
| Pulse glow | 2.4s | ease-in-out | Sync indicator breathing glow |
| Drag overlay | N/A | N/A | Opacity 0.4 while dragging |

**Reduced motion:** All animations disabled when `prefers-reduced-motion: reduce` is set.

## Accessibility

- **Touch targets:** Minimum 44x44px on touch devices (`@media (hover: none) and (pointer: coarse)`)
- **Focus visible:** `2px solid rgba(16, 185, 129, 0.85)` with `outline-offset: 2px`, `border-radius: 14px`
- **Skip link:** "İçeriğe geç" — visible on focus, positioned top-left
- **ARIA:** `aria-busy`, `aria-live="polite"` on loading states; `aria-pressed` on toggle buttons; `aria-label` on icon buttons
- **Color contrast:** All text meets WCAG AA (ink on bg surfaces)
- **Safe area:** `env(safe-area-inset-bottom)` respected on mobile bottom nav
